# Session logs

Every task done in this frontend-web repo is recorded here so the **mobile app repo can be updated to match later**.

## How to log a task

1. Create a folder: `session/<date>/<task-name>/` (e.g. `session/2026-07-10/add-asset-filter/`).
2. Copy [TEMPLATE.md](TEMPLATE.md) into it as `README.md` and fill it in.
3. Record: what the task is, which files/features changed, key decisions, and what needs porting to mobile.

## Structure

```
session/
  TEMPLATE.md              # copy this for each task
  README.md                # this file
  2026-07-10/
    add-asset-filter/
      README.md
```
